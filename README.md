# FAST Missions Bot

Автоматизация миссий с [work.fast.xyz](https://work.fast.xyz/): бот проходит агентский
checkout-путь на [shop.fast.xyz](https://shop.fast.xyz/) для каждого мерчанта и снимает
6 обязательных скриншотов шагов миссии.

## Что делает бот (по шагам миссии)

1. **01-homepage** — открывает shop.fast.xyz (cookie-баннер: Essential Only).
2. **02-chatbot-open** — активирует ассистента Pif (поле «Search for something to buy»).
3. **03-search-results** — отправляет точный промпт мерчанта, ждёт ответ.
4. **04-product-detail** — выбирает первый подходящий товар.
5. **05-checkout** — начинает оформление. Никакие платёжные данные не вводятся.
6. **06-stop-before-payment** — останавливается в последней безопасной точке до оплаты.

Плюс `meta.json` (device info) и `screenshots/summary.json` по всем миссиям.

## Безопасность

- Бот **никогда** не вводит платёжные данные (guard `FORBIDDEN_INPUT` с проверкой перед финальным скриншотом).
- Бот **никогда** не нажимает Pay now / Place order / Complete purchase (`FORBIDDEN_CLICK`).
- Опциональный бонус-шаг «complete purchase» намеренно не реализован.

## Запуск

```bash
npm install
npx playwright install --with-deps chromium

# все 13 миссий
npm start

# одна миссия по slug (см. config/merchants.json)
node src/bot.js amazon
```

Скриншоты попадают в `screenshots/<slug>/`.

## Ограничения

- Миссии требуют скриншот «с видимой адресной строкой» — в headless-режиме браузерного
  UI нет, поэтому для финальной сдачи страницу 01 может понадобиться переснять в обычном
  браузере.
- На момент написания все миссии на work.fast.xyz достигли лимита (50/50), приём
  сабмитов закрыт. Бот готовит материалы на случай открытия новых слотов.
- Загрузка скриншотов на work.fast.xyz не автоматизирована сознательно: сабмит связан с
  аккаунтом и капами, его лучше делать вручную.

## Конфиг

`config/merchants.json` — список мерчантов: slug, сайт, страна, режимы checkout и
точный промпт миссии.
